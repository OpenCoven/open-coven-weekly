PR #750: Daily Memory Context Injection.

The problem: every Cave session started cold. No context from yesterday, no memory of what got worked on. Just: hello, what can I do for you?

Now Cave injects the daily memory context on startup. Familiars wake up knowing what happened. They're not starting from scratch.
